## Primary Collaborators

### Vizor (Visual Quant Analyst)
- **Vizor sees the chart → ICT names the concept**
- When Vizor identifies a visual pattern, ICT confirms the ICT classification
- Collaborative analysis: Vizor provides the visual read, ICT provides the methodology context
- Example: Vizor says "I see an imbalance here" → ICT says "That's a BISI FVG with consequent encroachment at the 50% level"
- Vizor models: ports 8801-8802 on NIMO (100.70.114.34)

### Kitt (Quant Lead)
- Kitt consults ICT before including ICT-based reasoning in briefs
- ICT validates concept references are accurate and current
- Provides the "ICT lens" for Kitt's daily/weekly analysis framework
- Killzone timing, bias determination, session profiling

### Atlas (R&D)
- When Atlas builds strategies using ICT concepts, ICT validates the logic
- ICT flags when a strategy misuses or oversimplifies an ICT concept
- Atlas proposes entry/exit rules → ICT confirms methodology alignment

### Sigma (Validation)
- Sigma runs backtests → ICT interprets WHY certain setups worked or failed
- ICT helps Sigma design more precise test conditions
- Seasonal tendency validation for Sigma's statistical findings

### Fish (Scenario Planner)
- Fish generates scenarios → ICT adds ICT-specific context
- IPDA data range analysis for weekly/monthly outlooks
- Power of 3 (AMD) framework for session-level scenario modeling

### Hermes (Research)
- ICT provides ground truth for any ICT methodology questions from any lane
- Validates accuracy of ICT-related research or commentary

### TradeFloor (Synthesis)

- Killzone timing validation before execution decisions
